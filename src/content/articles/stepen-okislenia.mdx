## Как определить степень окисления?

Степень окисления необходима для электронного баланса в ЕГЭ, ОГЭ, а также в контрольной по химии в школе. Электронный баланс для окислительно-восстановительных реакций (ОВР) считается необходимым пунктом в заданиях ЕГЭ, чтобы расставить коэффициенты в уравнении реакции ОВР. Для его написания необходимо знать правила расставления степеней окисления. Что это такое? Как определить степень окисления?

Степень окисления - важная тема для ЕГЭ по химии. Полный план подготовки к экзамену читайте в статье [Как подготовиться к ЕГЭ по химии с нуля](/media/articles/kak-podgotovitsya-k-ege-po-himii-s-nulya).

## Что такое степень окисления простыми словами

<Quote author="Определение">
  Степень окисления - это условный заряд атома, который атом «получил бы» при
  полном ионном характере связей. Чаще всего она совпадает с валентностью (без
  знака), но не всегда.
</Quote>

Например, в азотной кислоте HNO<sub>3</sub> степень окисления азота +5, а валентность равна IV (поэтому в определении часто говорят, что это **условный** заряд атома).

Для определения положительного/отрицательного знака и цифр используют правило:

- если атом **отдаёт** электроны, он приобретает **положительный** заряд;
- если атом **принимает** электроны, он приобретает **отрицательный** заряд.

Например, Cu в степени окисления 0 отдаёт 2 электрона и «принимает» степень окисления +2. Кратко это записывают так:  
Cu<sup>0</sup> − 2e → Cu<sup>+2</sup>

Фосфор в степени окисления 0 принимает 3 электрона и приобретает степень окисления −3. Кратко:  
P<sup>0</sup> + 3e → P<sup>−3</sup>

### Электроотрицательность

Положительный и отрицательный заряд помогает определить электроотрицательность атома. **Электроотрицательность** - это способность атома притягивать электроны. Чем выше электроотрицательность, тем легче атому притянуть электрон и принять отрицательную степень окисления. Чем ниже - тем легче отдать электрон и получить положительную степень окисления.

>

| ------------- | ------------------------ | ------------- | ------------------------ |
| Фтор (F)      | 4,0                      | Водород (H)   | 2,1                      |
| Кислород (O)  | 3,5                      | Фосфор (P)    | 2,1                      |
| Хлор (Cl)     | 3,0                      | Углерод (C)   | 2,5                      |
| Азот (N)      | 3,0                      | Сера (S)      | 2,5                      |
| Бром (Br)     | 2,8                      | Кремний (Si)  | 1,8                      |
| Йод (I)       | 2,5                      | Бор (B)       | 2,0                      |
| Селен (Se)    | 2,4                      | Литий (Li)    | 1,0                      |
| Серебро (Ag)  | 1,9                      | Калий (K)     | 0,8                      |

</Table>

<Banner type="info" title="Важно знать">
  По таблице видно, что электроотрицательность неметаллов выше, чем у металлов,
  поэтому металлы чаще проявляют **положительную** степень окисления.
</Banner>

Например, в соединении SO<sub>2</sub> кислород (χ = 3,5) принимает отрицательную степень окисления, сера (χ = 2,5) - положительную. На знак в соединении влияет различие электроотрицательностей.

## Правила определения степени окисления

<Banner type="success" title="Базовое правило">
  Степень окисления **простых веществ** равна 0. Например: Cl<sub>2</sub>
  <sup>0</sup>, S<sup>0</sup>, Cr<sup>0</sup> и т. д.
</Banner>

**Постоянная** степень окисления бывает у следующих элементов:

| Атом (символ)        | Постоянная степень | Примеры веществ |
| -------------------- | ------------------ | --------------- |
| ЩМ (IA) + Ag         | +1                 | NaCl, KNO₃      |
| II группа (кроме Hg) | +2                 | MgBr₂, ZnSO₄    |
| Al                   | +3                 | Al₂(SO₄)₃       |

**Переменную** степень окисления имеют следующие элементы:

<Table

### Максимальная и минимальная степени окисления

<Banner type="info" title="Правило максимальной степени">
  **Максимальная** степень окисления обычно равна номеру группы, но есть
  исключения:
</Banner>

<Table
  title="Максимальные степени окисления"
  caption="Таблица 4. Исключения для максимальной степени окисления (запомните: КОФЕ - CuOFFe)"
  compact={true}
>

| Атом (символ) | Макс. степень | Примеры               |
| ------------- | ------------- | --------------------- |
| Cu            | +2            | CuSO₄, CuO            |
| O             | +2            | OF₂                   |
| F             | 0             | F₂                    |
| Fe            | +6            | K₂FeO₄ (феррат калия) |

</Table>

<Banner type="success" title="Лайфхак для запоминания">
  Представленные атомы составляют слово «КОФЕ» - **CuOFFe**. Запомните это
  слово, и вы всегда вспомните исключения!
</Banner>

**Минимальная** степень окисления:

- У неметаллов равна (№ группы − 8)
- У металлов равна 0 (так как они обычно проявляют положительные степени окисления)

Исключения среди неметаллов:

- Бор (B), минимальная степень −3
- Водород (H) - −1

## Алгоритм определения степени окисления. Примеры

<Banner type="info" title="Ключевое правило">
  Сумма всех степеней окисления в молекуле = 0. Для ионов сумма степеней равна
  заряду иона.
</Banner>

Зная постоянные и переменные степени окисления, можно находить неизвестные из условия суммы степеней.

### Примеры для нейтральных молекул

**HNO<sub>3</sub>** (азотная кислота)  
Определим известные степени: H - +1 (с неметаллом), O - −2 (не пероксид и не OF<sub>2</sub>). Осталось найти азот x.

Составим уравнение:  
+1 + x + (−2)·3 = 0 → 1 + x − 6 = 0 → **x = +5**

**K<sub>2</sub>SO<sub>4</sub>**  
K: +1, O: −2, S: x  
(+1)·2 + x + (−2)·4 = 0 → 2 + x − 8 = 0 → **x = +6**

**Ba<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>**  
Ba: +2, P: x, O: −2  
(+2)·3 + x·2 + (−2)·4·2 = 0 → **x = +5**

**Na<sub>2</sub>O<sub>2</sub>** (пероксид)  
Na: +1, O: x  
(+1)·2 + 2x = 0 → **x = −1**

**KMnO<sub>4</sub>**  
K: +1, O: −2, Mn: x  
+1 + x + (−2)·4 = 0 → **x = +7**

### Примеры для ионов

**ClO<sub>4</sub><sup>−</sup>**  
Cl: x, O: −2  
x + (−2)·4 = −1 → **x = +7**

**SO<sub>4</sub><sup>2−</sup>**  
S: x, O: −2  
x + (−2)·4 = −2 → **x = +6**

**Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup>**  
Cr: x, O: −2  
2x + (−2)·7 = −2 → **x = +6**

## Типичные ошибки

<Banner type="warning" title="Ошибка №1: Путаница знака и числа">
  Не путайте **степень окисления** с **зарядом**. - **Заряд:** число + знак (2−,
  1−, 3+) - **Степень окисления:** знак + число (+2, −3, −1)
</Banner>

<Banner type="warning" title="Ошибка №2: Степень ≠ Валентность">
  Степень окисления **не равна** валентности. Валентность - количество связей у
  атома; степень окисления - **условный** заряд.
</Banner>

<Banner type="warning" title="Ошибка №3: Забыли про сумму">
  Помните: сумма степеней окисления в молекуле всегда 0, а у ионов - равна
  **заряду** иона.
</Banner>

## Часто задаваемые вопросы

<FAQ question="Как определить степень окисления?" open={true}>
  Краткий алгоритм: определите элементы с постоянной степенью и запишите её →
  обозначьте неизвестную степень за переменную x → составьте линейное уравнение
  и приравняйте к 0 (или к заряду иона) → найдите x.
</FAQ>

<FAQ question="Чем валентность отличается от степени окисления?">
  Валентность - число связей от атома в молекуле (римские цифры: VI, V, III).
  Степень окисления - условный заряд атома при образовании связей (арабские
  цифры со знаком: +3, −2, −7).
</FAQ>

<FAQ question="Где на ЕГЭ нужна степень окисления?">
  Встречается в заданиях №3 (постоянные/переменные степени), №19 (ОВ-свойства
  атомов/ионов), №29 (ОВР: электронный баланс, степени окисления,
  окислитель/восстановитель). Без знаний степеней окисления решить эти задания
  невозможно. Для правильного названия веществ важно знать не только степень
  окисления, но и номенклатуру. Подробнее читайте в статье [Как называть
  вещества в химии](/media/articles/kak-nazivat-veschestva-v-himii).
</FAQ>

<FAQ question="Что такое электроотрицательность?">
  Электроотрицательность - это способность атома притягивать электроны. Чем выше
  электроотрицательность, тем легче атому принять отрицательную степень
  окисления. Самая высокая электроотрицательность у фтора (4,0), самая низкая -
  у калия (0,8).
</FAQ>

<FAQ question="Как запомнить исключения для максимальной степени?">
  Запомните слово КОФЕ (CuOFFe): Cu (+2), O (+2), F (0), Fe (+6). Эти элементы
  не следуют правилу «максимальная степень = номер группы».
</FAQ>
