## Как определить степень окисления?

Этот показатель необходим для электронного баланса в ЕГЭ, ОГЭ, а также в контрольной по химии в школе. Электронный баланс для окислительно-восстановительных реакций (ОВР) — обязательный пункт в заданиях ЕГЭ для расстановки коэффициентов. Для его составления необходимо знать правила расстановки зарядов атомов. Что это такое?

Это важная тема для ЕГЭ по химии. Правила расстановки и ОВР подробно разбираются в [курсе по неорганике](/neorganika/).

<Banner type="info" title="TL;DR: степень окисления для ЕГЭ и задания №3">
- **Что такое степень окисления?** Это условный заряд атома в веществе.

- **Как определить степень окисления быстро?** Сначала поставьте постоянные значения: щелочные металлы +1, элементы II группы +2, Al +3, F −1, O обычно −2, H обычно +1. Потом используйте правило суммы: в молекуле сумма равна 0, в ионе — заряду иона.

- **Как решать задание №3 на ЕГЭ?** Сначала поймите, о чём спрашивают: постоянная степень, высшая, низшая, возможная в соединении или разность между ними. Потом отсейте элементы с постоянными значениями и проверьте исключения.

- **Базовые паттерны:** у металлов в соединениях нет отрицательных степеней окисления; у неметаллов часто есть и отрицательные, и положительные; у кислорода обычно −2, но в пероксидах −1, а в OF<sub>2</sub> +2; у водорода обычно +1, а в гидридах металлов −1.

- **Быстрые переходы:** [алгоритм](/academy/obschaya-himiya/stepen-okislenia/#algoritm-opredeleniya-stepeni-okisleniya-primery), [популярные степени окисления на ЕГЭ](/academy/obschaya-himiya/stepen-okislenia/#naibolee-populyarnye-stepeni-okisleniya-elementov-na-ege), [разбор заданий ЕГЭ №3](/academy/obschaya-himiya/stepen-okislenia/#razbor-zadaniy-ege-zadanie-3).

</Banner>

## Что такое степень окисления простыми словами

<Quote author="Определение">
  Степень окисления - это условный заряд атома, который атом «получил бы» при
  полном ионном характере связей. Чаще всего она совпадает с валентностью (без
  знака), но не всегда.
</Quote>

Основы темы — знаки атомов и валентность — закладываются в [курсе по общей химии](/obschaya/).

Например, в азотной кислоте HNO<sub>3</sub> это значение для азота равно +5, а валентность — IV (поэтому в определении говорят именно об **условном** характере).

Для определения положительного/отрицательного знака и цифр используют правило:

- если атом **отдаёт** электроны, его знак становится **положительным**;
- если атом **принимает** электроны, его знак становится **отрицательным**.

Например, Cu(0) отдаёт 2 электрона и получает значение +2. Кратко:
Cu<sup>0</sup> − 2e → Cu<sup>+2</sup>

Фосфор P(0) принимает 3 электрона и получает значение −3. Кратко:
P<sup>0</sup> + 3e → P<sup>−3</sup>

### Электроотрицательность

Знак атома (+/-) помогает определить электроотрицательность. **Электроотрицательность** — это способность атома притягивать электроны. Чем она выше, тем легче атому принять электрон и получить отрицательное значение. Чем ниже — тем легче отдать электрон и получить положительное.

| ------------- | ------------------------ | ------------- | ------------------------ |
| Фтор (F)      | 4,0                      | Водород (H)   | 2,1                      |
| Кислород (O)  | 3,5                      | Фосфор (P)    | 2,1                      |
| Хлор (Cl)     | 3,0                      | Углерод (C)   | 2,5                      |
| Азот (N)      | 3,0                      | Сера (S)      | 2,5                      |
| Бром (Br)     | 2,8                      | Кремний (Si)  | 1,8                      |
| Йод (I)       | 2,5                      | Бор (B)       | 2,0                      |
| Селен (Se)    | 2,4                      | Литий (Li)    | 1,0                      |
| Серебро (Ag)  | 1,9                      | Калий (K)     | 0,8                      |

</Banner>

## Алгоритм определения степени окисления. Примеры

<Banner type="info" title="Ключевое правило">
  Алгебраическая сумма всех значений в молекуле = 0. Для ионов сумма равна
  заряду иона.
</Banner>

Зная постоянные и переменные значения, можно находить неизвестные из условия этой суммы.

### Примеры для нейтральных молекул

**HNO<sub>3</sub>** (азотная кислота)  
Определим известные значения: H — +1 (с неметаллом), O — −2 (не пероксид и не OF<sub>2</sub>). Осталось найти азот x.

Составим уравнение:  
+1 + x + (−2)·3 = 0 → 1 + x − 6 = 0 → **x = +5**

**K<sub>2</sub>SO<sub>4</sub>**  
K: +1, O: −2, S: x  
(+1)·2 + x + (−2)·4 = 0 → 2 + x − 8 = 0 → **x = +6**

**Ba<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>**  
Ba: +2, P: x, O: −2  
(+2)·3 + x·2 + (−2)·4·2 = 0 → **x = +5**

**Na<sub>2</sub>O<sub>2</sub>** (пероксид)  
Na: +1, O: x  
(+1)·2 + 2x = 0 → **x = −1**

**KMnO<sub>4</sub>**  
K: +1, O: −2, Mn: x  
+1 + x + (−2)·4 = 0 → **x = +7**

### Примеры для ионов

**ClO<sub>4</sub><sup>−</sup>**  
Cl: x, O: −2  
x + (−2)·4 = −1 → **x = +7**

**SO<sub>4</sub><sup>2−</sup>**  
S: x, O: −2  
x + (−2)·4 = −2 → **x = +6**

**Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup>**  
Cr: x, O: −2  
2x + (−2)·7 = −2 → **x = +6**

Эта тема нужна не только для ОВР: по степеням окисления проще разбирать соли, их ионы и последующий [гидролиз солей](/academy/obschaya-himiya/hydroliz-solei-na-ege-po-himii/), особенно в примерах с Al³⁺, Fe³⁺, карбонатами и фосфатами. А для реакций двух солей, где одновременно появляются осадок и газ, полезно отдельно разобрать [совместный гидролиз](/academy/neorganicheskaya-himiya/sovmestnii-hydrolyz/).

## Разбор заданий ЕГЭ, задание №3

В ЕГЭ по химии задание №3 часто проверяет именно понимание степеней окисления: где элемент может иметь только постоянное значение, где есть исключения, а где нужно сравнить высшую и низшую степени окисления. Ниже — подробные разборы типовых формулировок.

<Banner
  type="success"
  title="Как быстро решать задание №3 на степень окисления"
>
- Сначала определяйте, о каких степенях окисления спрашивают: высшая, низшая, постоянная, возможная в соединениях или в простом веществе.

- Для неметаллов полезно помнить максимум по группе и типичные исключения.

- Для металлов чаще всего важно знать, есть ли у них только положительные степени окисления и бывает ли ноль в простом веществе.

- Если в условии фигурируют анионы, сразу пользуйтесь правилом суммы степеней окисления.

</Banner>

### Задание 1. Разность между высшей и низшей степенями окисления равна 2

Из числа указанных в ряду элементов выберите два элемента, у каждого из которых разность между значениями его высшей и низшей степеней окисления равна 2.

1. V
2. O
3. H
4. F
5. Mg

**Решение**

Смотрим на каждый элемент отдельно.

- **V**: высшая степень окисления +5, низшая −3. Разность равна 8. Не подходит.
- **O**: низшая степень окисления −2, высшая +2 (в соединении OF<sub>2</sub>). Разность равна 4. Не подходит.
- **H**: низшая степень окисления −1 (в гидридах металлов), высшая +1 (в соединениях с неметаллами). Разность равна 2. Подходит.
- **F**: фтор в соединениях имеет степень окисления −1, а в простом веществе F<sub>2</sub> — 0. Разность равна 1. Не подходит.
- **Mg**: магний в соединениях обычно имеет +2, а в простом веществе Mg — 0. Разность равна 2. Подходит.

**Ответ: 3, 5.**

### Задание 2. Одинаковая степень окисления в анионах состава RO<sub>x</sub><sup>2−</sup>

Из числа указанных в ряду элементов выберите два элемента, которые в кислородсодержащих анионах состава RO<sub>x</sub><sup>2−</sup> могут проявлять одинаковую степень окисления.

1. K
2. Be
3. Si
4. Ca
5. Zn

**Решение**

В таких заданиях нужно не угадывать, а вспоминать реальные кислородсодержащие анионы.

- **Be** образует анион BeO<sub>2</sub><sup>2−</sup>. Тогда степень окисления бериллия: x + 2·(−2) = −2, значит x = +2.
- **Zn** образует анион ZnO<sub>2</sub><sup>2−</sup>. Тогда x + 2·(−2) = −2, значит x = +2.
- **Si** обычно рассматривают в силикатах, например SiO<sub>3</sub><sup>2−</sup>, где у кремния уже +4.
- **K** и **Ca** в школьной химии как центральные атомы кислородсодержащих анионов такого типа не рассматриваются.

Значит, одинаковую степень окисления в подходящих анионах имеют бериллий и цинк: у обоих это **+2**.

**Ответ: 2, 5.**

### Задание 3. Элемент может иметь и −1, и +1

Из числа указанных в ряду химических элементов выберите два, которые в соединениях могут проявлять степень окисления как −1, так и +1.

1. Mg
2. O
3. Cl
4. K
5. Cu

**Решение**

- **Mg** в соединениях имеет +2. Значения −1 и +1 для него нехарактерны.
- **O** может иметь **−1** в пероксидах и **+1** в соединении O<sub>2</sub>F<sub>2</sub>. Подходит.
- **Cl** может иметь **−1** в хлоридах и **+1** в HClO или солях хлорноватистой кислоты. Подходит.
- **K** имеет постоянную степень окисления +1, но −1 не проявляет.
- **Cu** может иметь +1 и +2, но −1 для него нехарактерна.

**Ответ: 2, 3.**

### Задание 4. Элементы не проявляют положительную степень окисления

Из указанных в ряду элементов выберите два элемента, которые не проявляют положительной степени окисления.

1. H
2. O
3. Zn
4. F
5. He

**Решение**

- **H** проявляет положительную степень окисления +1 в большинстве соединений с неметаллами. Не подходит.
- **O** может иметь положительную степень окисления, например в OF<sub>2</sub>. Не подходит.
- **Zn** проявляет положительную степень окисления +2. Не подходит.
- **F** как самый электроотрицательный элемент положительных степеней окисления не проявляет. Подходит.
- **He** как инертный газ в школьной химии рассматривают как элемент, не проявляющий положительных степеней окисления. Подходит.

**Ответ: 4, 5.**

### Задание 5. Высшая степень окисления не совпадает с номером группы

Из указанных в ряду элементов выберите два элемента, высшая степень окисления которых не совпадает с номером группы.

1. N
2. O
3. Cu
4. Br
5. S

**Решение**

Для таких заданий удобно сразу вспоминать исключения из правила максимальной степени окисления. Они как раз разобраны выше в блоке [«Максимальная и минимальная степени окисления»](/academy/obschaya-himiya/stepen-okislenia/#maksimalnaya-i-minimalnaya-stepeni-okisleniya).

- **N**: высшая степень окисления +5, что соответствует номеру группы. Не подходит.
- **O**: для кислорода высшая степень окисления не +6, а **+2**. Это исключение. Подходит.
- **Cu**: для меди максимальная степень окисления **+2**, а не значение по номеру группы. Это тоже исключение. Подходит.
- **Br**: высшая степень окисления +7, совпадает с номером группы. Не подходит.
- **S**: высшая степень окисления +6, совпадает с номером группы. Не подходит.

**Ответ: 2, 3.**

## Типичные ошибки

<Banner type="warning" title="Ошибка №1: Путаница знака и числа">
- **Заряд**: число + знак (2−, 1−, 3+).

- **Степень окисления**: знак + число (+2, −3, −1).

</Banner>

<Banner type="warning" title="Ошибка №2: Степень ≠ Валентность">
  Степень окисления **не равна** валентности. Валентность - количество связей у
  атома; степень окисления - **условный** заряд.
</Banner>

<Banner type="warning" title="Ошибка №3: Забыли про сумму">
  Помните: алгебраическая сумма в молекуле всегда 0, а у ионов — равна
  **заряду** иона.
</Banner>

## Часто задаваемые вопросы

<FAQ question="Как определить степень окисления?" open={true}>
  Краткий алгоритм: определите элементы с известным значением и запишите его →
  обозначьте неизвестное за переменную x → составьте линейное уравнение и
  приравняйте к 0 (или к заряду иона) → найдите x.
</FAQ>

<FAQ question="Чем валентность отличается от степени окисления?">
  Валентность - число связей от атома в молекуле (римские цифры: VI, V, III).
  Степень окисления - условный заряд атома при образовании связей (арабские
  цифры со знаком: +3, −2, −7).
</FAQ>

<FAQ question="Где на ЕГЭ нужна степень окисления?">
  Встречается в заданиях №3 (постоянные/переменные значения), №19 (ОВ-свойства
  атомов/ионов), №29 (ОВР: электронный баланс, окислитель/восстановитель). Без
  знания этой темы решить эти задания невозможно. Для правильного названия
  веществ важно знать не только этот показатель, но и номенклатуру. Подробнее
  читайте в статье [Как называть вещества в
  химии](/academy/neorganicheskaya-himiya/kak-nazivat-veschestva-v-himii/).
</FAQ>

<FAQ question="Что такое электроотрицательность?">
  Электроотрицательность — это способность атома притягивать электроны. Чем она
  выше, тем легче атому получить отрицательное значение. Самая высокая — у фтора
  (4,0), самая низкая — у калия (0,8).
</FAQ>

<FAQ question="Как запомнить исключения для максимальной степени?">
  Запомните слово КОФЕ (CuOFFe): Cu (+2), O (+2), F (0), Fe (+6). Эти элементы
  не следуют правилу «максимальная степень = номер группы».
</FAQ>
