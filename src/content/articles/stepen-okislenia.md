## Как определить степень окисления?

Степень окисления необходима для электронного баланса в ЕГЭ, ОГЭ, а также в контрольной по химии в школе. Электронный баланс для окислительно-восстановительных реакций (ОВР) считается необходимым пунктом в заданиях ЕГЭ, чтобы расставить коэффициенты в уравнении реакции ОВР. Для его написания необходимо знать правила расставления степеней окисления. Что это такое? Как определить степень окисления?

## Что такое степень окисления простыми словами

Степень окисления — это условный заряд атома, который атом «получил бы» при полном ионном характере связей. Чаще всего она совпадает с валентностью (без знака), но не всегда. Например, в азотной кислоте HNO<sub>3</sub> степень окисления азота +5, а валентность равна IV (поэтому в определении часто говорят, что это **условный** заряд атома).

Для определения положительного/отрицательного знака и цифр используют правило:

- если атом **отдаёт** электроны, он приобретает **положительный** заряд;
- если атом **принимает** электроны, он приобретает **отрицательный** заряд.

Например, Cu в степени окисления 0 отдаёт 2 электрона и «принимает» степень окисления +2. Кратко это записывают так:  
Cu<sup>0</sup> + 2e → Cu<sup>+2</sup>

Фосфор в степени окисления 0 принимает 3 электрона и приобретает степень окисления −3. Кратко:  
P<sup>0</sup> − 3e → P<sup>−3</sup>

Положительный и отрицательный заряд помогает определить электроотрицательность атома. **Электроотрицательность** — это способность атома притягивать электроны. Чем выше электроотрицательность, тем легче атому притянуть электрон и принять отрицательную степень окисления. Чем ниже — тем легче отдать электрон и получить положительную степень окисления. Исключения среди неметаллов: бор (B), минимальная степень −3; водород (H) — −1.

## Алгоритм определения степени окисления. Примеры

Зная постоянные и переменные степени окисления, можно находить неизвестные из условия <b>суммы степеней</b>, равной 0 для нейтральных молекул.

Допустим, дана молекула азотной кислоты — HNO<sub>3</sub>.  
Определим известные степени: H — +1 (с неметаллом), O — −2 (не пероксид и не OF<sub>2</sub>). Осталось найти азот x.

**Правило:** сумма всех степеней окисления в молекуле = 0. Составим уравнение:

+1 + x + (−2)·3 = 0 → 1 + x − 6 = 0 → **x = +5**

Ещё примеры:

**K<sub>2</sub>SO<sub>4</sub>**  
K: +1, O: −2, S: x  
(+1)·2 + x + (−2)·4 = 0 → 2 + x − 8 = 0 → **x = +6**

**Ba<sub>3</sub>(PO<sub>4</sub>)<sub>2</sub>**  
Ba: +2, P: x, O: −2  
(+2)·3 + x·2 + (−2)·4·2 = 0 → **x = +5**

**Na<sub>2</sub>O<sub>2</sub>** (пероксид)  
Na: +1, O: x  
(+1)·2 + 2x = 0 → **x = −1**

**KMnO<sub>4</sub>**  
K: +1, O: −2, Mn: x  
+1 + x + (−2)·4 = 0 → **x = +7**

Если молекула нейтральна, сумма степеней = 0. Для **ионов** сумма степеней равна **заряду иона**.

Примеры ионов:

**ClO<sub>4</sub><sup>−</sup>**  
Cl: x, O: −2  
x + (−2)·4 = −1 → **x = +7**

**SO<sub>4</sub><sup>2−</sup>**  
S: x, O: −2  
x + (−2)·4 = −2 → **x = +6**

**Cr<sub>2</sub>O<sub>7</sub><sup>2−</sup>**  
Cr: x, O: −2  
2x + (−2)·7 = −2 → **x = +6**

## Типичные ошибки

a) Не путайте **степень окисления** с **зарядом**.  
Заряд: число + знак (2−, 1−, 3+).  
Степень окисления: знак + число (+2, −3, −1).

б) Степень окисления **не равна** валентности. Валентность — количество связей у атома; степень окисления — **условный** заряд.

в) Помните: сумма степеней окисления в молекуле всегда 0, а у ионов — равна **заряду** иона.

## Часто задаваемые вопросы

**1. Как определить степень окисления?**  
Краткий алгоритм:

- определить элементы с постоянной степенью и записать её;
- обозначить неизвестную степень за переменную x;
- составить линейное уравнение и приравнять к 0 (или к заряду иона);
- найти x.

**2. Чем валентность отличается от степени окисления?**  
Валентность — число связей от атома в молекуле (римские цифры: VI, V, III).  
Степень окисления — условный заряд атома при образовании связей (арабские цифры со знаком: +3, −2, −7).

**3. Где на ЕГЭ нужна степень окисления?**  
Встречается в заданиях №3 (постоянные/переменные степени), №19 (ОВ-свойства атомов/ионов), №29 (ОВР: электронный баланс, степени окисления, окислитель/восстановитель). Без знаний степеней окисления решить эти задания невозможно.

Если вы выучите правила и потренируетесь, задачи на степень окисления будут даваться легко. Правила пригодятся на ЕГЭ и ОГЭ — по ним вы решите задания на ОВР и степени окисления.
